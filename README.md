Project Description
A Machine Learning project that classifies text reviews as Positive or Negative. This project demonstrates the fundamental concepts of Natural Language Processing (NLP) and supervised learning.
Key Technical Steps
Text Vectorization: Converting raw text into numerical features using CountVectorizer (Bag of Words model).
Model Selection: Implemented a Multinomial Naive Bayes classifier, known for its efficiency in text classification tasks.
Data Splitting: Used train_test_split to ensure the model generalizes well to unseen data.
Technologies Used:
Python
Pandas (Data Manipulation)
Scikit-learn (Machine Learning Library)
